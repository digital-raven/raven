# Patterns, cores, and attachments
These are re-usable nix modules created for Raven.

To create a full Raven pattern, choose a core and utilities from the
directories here.

- cores: Import `raven-core`
- terminal: High-quality terminal utilities and home-manager customizations.
- graphical: General graphical applications. eg. firefox, gimp,  
- hyprland: Raven's hyprland offering.

The `raven-*` example patterns show a proper demonstration.
